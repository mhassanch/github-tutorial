Github using git

Pre requisities

  • Install Git

    Download git bash from

       http://git-scm.com/downloads

    and then install in computer to create repository in local machine.

  • Register on GitHub

    Go the following link to sign in or sign up on github.

       https://github.com/

  • Knowledge

    You just need the basic knowledge of coding to start with.

Create New Repository


  • Sign in to your github account.

  • On the home page, click on 'New Repository'.

  • Enter Repository name.

  • Set privacy policy as private or public.

  • Then click on Create Repository.
